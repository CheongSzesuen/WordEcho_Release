# WordEcho 公告推送系统上线

本次更新带来了全新的 **公告推送系统** 和 *更新检测机制*，以后重要通知会通过此系统直接推送到你的设备。

## 功能介绍

### 公告系统

- **自动弹窗**：启动 App 时自动拉取最新公告
- **锁定阅读**：部分重要公告会锁定 `N` 秒，确保你看完
- **不再提醒**：可选择对已读公告不再打扰
- **历史公告**：可在设置页面查看所有历史公告

### 更新检测

当检测到新版本时，会弹出更新对话框。你可以：
1. 点击「前往官网」下载最新版本
2. 或选择「稍后再说」（非强制更新时）

## 技术细节

这次我们在底层使用了 ~~旧的轮询方案~~ 新的推送架构：

```
Client → Gitee Raw → announcement.json → Filter → Display
```

同时支持 **版本号过滤** 和 *用户分组* 等高级特性。

> 注：当前为 Alpha 阶段，如有问题请通过 `关于页面` → `反馈` 提交。
