## 新增

- **更新日志总览**：关于页面新增「更新日志」入口，可查看各版本更新内容，点击展开加载对应 Markdown
- **公告总览**：关于页面新增「公告总览」入口，集中展示所有历史公告，点击展开查看详情
- **多镜像源容灾**：所有 GitHub 文件拉取（公告、更新、词书）接入 Gitee、jsDelivr 及多个代理镜像，任一源失败自动切换，国内访问更稳定
- **版本追踪体系**：新增 `updates.csv` 版本索引与 `updates/{版本}/{平台}.md` 目录结构，支持多平台（Android / Vela）独立日志，预留后续平台扩展

## 优化

- 词书数据下载支持多镜像重试，单个镜像不可用时自动切换，下载成功率提升
- 统一远程文件拉取与 CSV 解析逻辑，消除公告、更新服务间的重复代码
- CSV 解析支持转义双引号（`""`），字段含逗号时解析更稳健

## 修复

- 修复公告/更新数据在 GitHub raw 不可达时直接失败的问题，现可经镜像源兜底
